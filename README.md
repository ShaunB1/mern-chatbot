# MERN Chatbot
A fullstack web application that utilizes the OpenAI API to generate responses to the user's input.

## Features
* Account Signup
* User Authentication
* Cookies
* Password Hashing

This web application was created using MongoDB, ExpressJS, ReactJS, NodeJS
